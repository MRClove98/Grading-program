# Grading-program
This is a grading program where you can enter the amount of students taking the test and all of their names. You can also input the child's percentage and the program will automatically tell you their grades. This program will also tell you the highest scorer, lowest scorer and the average score. We made this program easy and fun to use!

## Features
-Input amount of students
-Input the student's name
-Input the percentage they got
-Give back accurate grades and tells you the highest scorer, lowest scorer and the average score
